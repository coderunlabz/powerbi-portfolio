# 📊 Análise de Vendas Globais

Dashboard analítico sobre performance de vendas globais com foco em decisão comercial e operacional.

## 🔍 Perguntas que o Dashboard Responde

1. Quais países geram mais receita e como se distribuem por prioridade de envio?
2. Quais categorias de produto lideram em volume de vendas?
3. Onde os descontos estão impactando a margem por subcategoria?
4. Quais países têm maior média de valor de venda?
5. Como a prioridade de envio varia por região geográfica?

## 📁 Fonte dos Dados

Dataset público do curso de Power BI — [Data Science Academy (DSA)](https://www.datascienceacademy.com.br/).  
Dados de vendas globais fictícios para fins educacionais.

## 🛠️ Ferramentas e Técnicas

- **Power BI Desktop** — construção do relatório
- **Power Query** — tratamento, limpeza e transformação dos dados
- **DAX** — métricas calculadas (totais, médias, percentuais)
- **Mapa Preenchido** — visualização geográfica de vendas por país
